Для выполнения задания по варианту XX, нам необходимо реализовать алгоритм выделения отличий изображений по форме (морфологический анализ по Пытьеву) со следующими параметрами:Цветовое пространство: CIELAB ($L, a, b$).Метод: Скользящее окно (локальный анализ).Метрика $\delta$: Сумма квадратов минимальных размеров квадратов $r_c$ для каждого канала: $\delta = r_L^2 + r_a^2 + r_b^2$.Условие: Минимальный размер квадрата $r_c$ — это такая величина, при которой в облаке точек $(f_c(x), g_c(x))$ внутри окна можно провести график монотонно неубывающей функции $F_c$, проходящий через все квадраты со стороной $r_c$ и центрами в этих точках.Теоретическое обоснованиеСуть метода Пытьева заключается в том, что два изображения имеют одинаковую «форму», если яркости одного являются монотонным преобразованием другого: $g(x) = F(f(x))$. Если такое $F$ существует, то изображения морфологически эквивалентны.Величина $r$ (размер квадрата) показывает, насколько сильно нужно «размыть» значения пикселей, чтобы условие монотонности выполнилось. Если $r$ велико, значит, в окне есть существенные структурные различия (например, появился новый объект), которые нельзя списать на простое изменение освещения (которое обычно моделируется монотонной функцией).
